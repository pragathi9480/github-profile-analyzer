
# GitHub Portfolio Analyzer & Enhancer

## Problem Statement
Recruiters often struggle to evaluate GitHub profiles due to poor documentation,
inconsistent activity, and unclear project impact — even when the candidate is skilled.

## Solution
A web-based tool that analyzes public GitHub profiles and generates an objective
GitHub Portfolio Score along with strengths, red flags, and actionable recommendations.

## Features
- GitHub Portfolio Score (0–100)
- Repository & activity analysis
- Strengths and red flags
- Actionable improvement suggestions
- Clean and simple user interface

## Tech Stack
- HTML
- CSS
- JavaScript
- GitHub REST API

## Scoring Criteria
- Repository Count (10)
- Documentation Quality (20)
- Recent Activity (20)
- Language Diversity (15)
- Project Impact (15)
- Consistency (20)

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Enter a GitHub profile URL
4. Click **Analyze Profile**
